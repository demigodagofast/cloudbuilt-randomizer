# Cloudbuilt Randomizer for Archipelago
This is a randomizer implementation for [Cloudbuilt](https://store.steampowered.com/app/262390/Cloudbuilt/) for use in [Archipelago](https://archipelago.gg/). It is implemented using [Manual for Archipelago](https://github.com/ManualForArchipelago/Manual) version "Stable - 20260319"

# Installation
Follow the guide [here](https://steamcommunity.com/sharedfiles/filedetails/?id=3746948768)

# Reporting Bugs
If you find any errors in the randomizer logic (impossible checks, etc) or world generation, please let me know by opening an issue.
